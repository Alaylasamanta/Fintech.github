# Fintech.github
PagaCerto é uma fintech criada para ajudar usuários a evitar atrasos em boletos, controlar entradas e saídas e melhorar a saúde financeira. O app oferece centralização de contas, alertas, agendamento via PIX, fluxo de caixa, poupança automática e investimentos simples. Este repositório inicia o protótipo com a tela de login.
